Projetos

A nossa história!

NERDigital

Este projeto combina um sensor impressão digital com um projeto já feito anteriormente. Através de um sensor de impressões digitais, o utilizador pode autenticar-se, permitindo que o sistema verifique a identidade e envie um sinal para o NeRDOOR responsável por controlar o motor da porta. Esse segundo sistema, conectado via wi-fi, recebe a autorização e aciona o mecanismo de abertura da porta.

NERDoor

Este projeto é um sistema automatizado de abertura de porta utilizando motores. A solução consiste na instalação de um servo, controlado por um microcontrolador, que aciona o mecanismo de abertura e fecho da porta. O sistema pode ser ativado por diferentes métodos, como cartão, via impressão digital e através smartphone.

NERDirection

Pretendemos desenvolver um projeto para facilitar a localização de salas nos departamentos da UA. O sistema interativo permitirá que os alunos encontrem salas de forma rápida. Utilizando tecnologias como mapas interativos e até realidade aumentada, os utilizadores poderão receber direções detalhadas para qualquer sala. Este projeto visa melhorar a experiência dos estudantes e visitantes, simplificando a navegação no departamento com uma solução tecnológica moderna e intuitiva.
